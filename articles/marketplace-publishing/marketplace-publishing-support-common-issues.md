# <a name="how-to-troubleshoot-common-azure-marketplace-publisher-issues"></a>Sposoby rozwiązywania typowych problemów z portalu Azure Marketplace wydawcy
W tym artykule wyjaśniono, jak rozwiązać lub uzyskać pomoc techniczną dla niektórych typowych scenariuszy Marketplace.

## <a name="credits-and-refunds"></a>Kredyty i zwroty
W niektórych sytuacjach może być konieczne żądania kredyt lub zwrot w imieniu klienta.  Microsoft Support pozwala zagwarantować, że żądania środki i zwrot klienta są traktowane jako i że odpowiedzi są dostępne w odpowiednim czasie.

Poniżej przedstawiono sposób żądania kredytowej lub zwrot w imieniu klienta.

> [!NOTE]
> Jeśli odbiorcy lub sprzedawcą Azure skontaktuje firmy Microsoft, aby zażądać zwrotu zapłaconej kwoty lub środki, zostaną zastosowane te same zasady używających dla naszych świadczenia usług Azure. Jeśli nasze zasady wskazywać zwrotu zapłaconej kwoty lub środki termin, firma Microsoft będzie wystawiać aktywne (i NAS do wystawiania w Twoim imieniu) zwrotu zapłaconej kwoty lub środki Azure odsprzedawcy lub klienta. Jeśli nasze zasady wskazuje, że zwrotu zapłaconej kwoty lub kredytowej jest termin, firma Microsoft może zaprosić klienta lub sprzedawcą Azure się bezpośrednio z Tobą skontaktować.
>
>

  ![1_RequestRefund][1]

1. Przejdź do strony pomocy technicznej, w tym miejscu: [https://support.microsoft.com/en-us/getsupport?tenant=ClassicCommercial&locale=en-us&supportregion=en-us&pesid=15635&sd=&oaspworkflow=start_1.0.0.0&wf=0& ccsid = 635562546535117884](https://support.microsoft.com/en-us/getsupport?tenant=ClassicCommercial&locale=en-us&supportregion=en-us&pesid=15635&sd=&oaspworkflow=start_1.0.0.0&wf=0&ccsid=635562546535117884).
2. W **typ problemu** wybierz opcję **Zarządzanie klientami**.
3. W **kategorii** wybierz opcję **Zatwierdź żądania lub poproszenia o zwrot klienta**.
4. Kliknij przycisk **żądanie uruchomienia**.

   ![2_RequestRefund][2]

1. Podaj informacje kontaktowe.
2. W **co to jest problem?** wprowadź następujące informacje:

   * Identyfikator subskrypcji wpływ na klientów
   * Nazwa klienta
   * Powinny mieć ilość środków
   * Okres środków
   * Data rachunku tego okresu
   * Przyczyna środków

Następnie otrzymasz wiadomość e-mail z potwierdzeniem w ciągu 24 godzin Otrzymaliśmy informacje. Po decyzji w odniesieniu do żądania, użytkownik jest informowany o wynikach.

## <a name="customer-subscription-validation"></a>Sprawdzanie poprawności subskrypcji klienta
W niektórych sytuacjach może być konieczne sprawdzić szczegóły subskrypcji dla jednego z klientów. Możesz skontaktować się Microsoft Support ułatwiające to. Microsoft Support pomogą upewnij się, że identyfikator subskrypcji i nazwisko, adres należy do znanego klienta.

  ![subscriptionvalidation][3]

* Klient może skontaktować się z Tobą w żądaniu pomocy technicznej.
* Może nie wiedzieć, jak sprawdzić, czy klient jest klienta oryginalnym oferty.
* Możesz zalogować się żądanie obsługi, aby zażądać weryfikacji subskrypcji klienta.
* Należy podać identyfikator subskrypcji klienta i nazwisko, adres.
* Microsoft skontaktuje się z należy podać adres do rozliczeń i identyfikator subskrypcji są znane nam potwierdzenie.

## <a name="how-do-i-add-or-delete-an-administrator-from-my-publishing-account"></a>Jak dodać lub usunąć administratora z mojego konta publikowania?
![manageadmins][8]

1. Przejdź do publikowania portalu.
2. Wybierz **wydawców**.
3. Wybierz **Administratorzy**.
4. Dodaj adres e-mail współadministratora, który chcesz dodać lub usunąć.
5. Wybierz pozycję **Zapisz**.
