# <a name="import-your-data-into-azure-machine-learning-studio-from-another-experiment"></a>Importowanie danych do usługi Azure Machine Learning Studio z poziomu innego eksperymentu
[!INCLUDE [import-data-into-aml-studio-selector](../../../includes/machine-learning-import-data-into-aml-studio.md)]

Będzie razy, kiedy należy podjąć pośrednich wyników z doświadczenia i użycie go jako część innego eksperymentu. Aby to zrobić, należy zapisać modułu jako zestawu danych:

1. Kliknij przycisk dane wyjściowe moduł, który ma zostać zapisany jako zestawu danych.
2. Kliknij przycisk **Zapisz jako zestawu danych**.
3. Po wyświetleniu monitu wprowadź nazwę i opis, który umożliwi łatwo identyfikować zestawu danych.
4. Kliknij przycisk **OK** znacznik wyboru.

Po zakończeniu Zapisz zestaw danych będą dostępne do użycia w ramach każdego doświadczenia w obszarze roboczym. Można znaleźć w **zapisane zestawów danych** listy na palecie modułów.
