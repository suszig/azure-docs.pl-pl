# <a name="overview-diagram-of-azure-machine-learning-studio-capabilities"></a>Diagram przeglądowy możliwości usługi Azure Machine Learning Studio
Diagram **Przegląd możliwości usługi Microsoft Azure Machine Learning Studio** przedstawia ogólny przegląd sposobu użycia usługi Machine Learning Studio w celu opracowania modelu analizy predykcyjnej i operacjonalizacji go w chmurze Azure.

Usługa [Azure Machine Learning Studio](https://studio.azureml.net/) dysponuje dużą liczbą algorytmów uczenia maszynowego oraz modułów, które ułatwiają wprowadzanie danych, ich przygotowywanie i wizualizację oraz opracowywanie wyników. Za pomocą tych składników można tworzyć eksperymenty analizy predykcyjnej, wykonywać ich iteracje, a następnie na ich podstawie uczyć modele.
Następnie wystarczy jedno kliknięcie, aby operacjonalizować model w chmurze Azure, dzięki czemu na jego podstawie można oceniać nowe dane.

Ten diagram przedstawia sposób dopasowania wszystkich tych elementów.

> [!NOTE]
> Aby uzyskać dodatkową pomoc w zakresie nawigowania i wybierania algorytmów uczenia maszynowego w usłudze Azure Machine Learning Studio, zobacz [Machine learning algorithm cheat sheet for Microsoft Azure Machine Learning Studio](algorithm-cheat-sheet.md) (Ściągawka dotycząca algorytmów uczenia maszynowego dla usługi Microsoft Azure Machine Learning Studio).
