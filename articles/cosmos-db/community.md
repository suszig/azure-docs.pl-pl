# <a name="community-portal"></a>Portal społeczności
## <a name="community-spotlight"></a>Spotlight społeczności
Daj nam Podwyższ poziom projektu! Pokazują świetny projektu, z którym pracujesz z bazy danych Azure rozwiązania Cosmos, generacji stosu usługi DocumentDB, a pomożemy udostępniać genius Twojego na świecie. Aby przesłać projektu, Wyślij do nas wiadomość e-mail na: [ askcosmosdb@microsoft.com ](mailto:askcosmosdb@microsoft.com).

### <a name="documentdb-lumenize"></a>Usługa documentdb-lumenize
*przez Larry Maccherone*

Agregacji (Grupuj według, Tabela przestawna i N-wymiarowej modułu) i czas serii przekształcenia jako procedury składowane w usłudze Azure rozwiązania Cosmos bazy danych DocumentDB interfejsu API.

Sprawdź go limit na [GitHub](https://github.com/lmaccherone/documentdb-lumenize) i [npm](https://www.npmjs.com/package/lumenize).

### <a name="documentdb-studio"></a>Studio usługi DocumentDB
*przez Ming Liu*

Klient podglądu/Eksploratora zarządzania dla usługi interfejsu API Azure rozwiązania Cosmos bazy danych usługi DocumentDB.

Sprawdź go limit na [GitHub](https://github.com/mingaliu/DocumentDBStudio).

### <a name="doqmentdb"></a>DoQmentDB
*przez Ariel Mashraki*

DoQmentDB jest klienta na podstawie promise Node.js, zapewniająca podobne bazy danych MongoDB warstwą bazy danych Azure rozwiązania Cosmos.

Sprawdź go limit na [GitHub](https://github.com/a8m/doqmentdb) i [npm](https://www.npmjs.com/package/doqmentdb).

### <a name="typescript-api"></a>TypeScript interfejsu API
*przez Jelmer Cormont*

Otokę klienta Node.js zapisywane na maszynie (działa zbyt w języku JavaScript zwykły). Obsługuje `async/await` i uproszczony interfejs API.

Sprawdź go limit na [GitHub](https://github.com/jcormont/documentdb-typescript) i [npm](https://www.npmjs.com/package/documentdb-typescript).

### <a name="swagger-rest-api-for-documentdb"></a>Swagger interfejsu API REST dla usługi DocumentDB
*przez Howard Edidin*

Plik Azure rozwiązania Cosmos DB REST API programu Swagger dla interfejsu API usługi DocumentDB, które można łatwo wdrożyć jako aplikację interfejsu API.

Sprawdź go limit na [GitHub](https://github.com/HEDIDIN/DocumentDB-REST/tree/master/DocumentDBRestApi).

### <a name="fluent-plugin-documentdb"></a>documentdb Fluent wtyczki
*przez Yoichi Kawasaki*

documentdb Fluent wtyczka jest dodatku Fluentd wyprowadzania do interfejsu API Azure rozwiązania Cosmos bazy danych usługi DocumentDB.

Sprawdź go limit na [GitHub](https://github.com/yokawasa/fluent-plugin-documentdb) i [rubygems](https://rubygems.org/gems/fluent-plugin-documentdb).

*Znajdź więcej Otwórz źródło projektów bazy danych Azure rozwiązania Cosmos w [GitHub](https://github.com/search?p=4&q=documentdb&type=Repositories).*
