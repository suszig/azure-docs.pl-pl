## <a name="alertlogic-log-manager"></a>Menedżer dzienników AlertLogic
Menedżer dzienników logiki alertu zbiera maszyny Wirtualnej, aplikacji i dzienniki platformy Azure, analizowania zabezpieczeń i przechowywania. Zbiera także dziennik aktywności platformy Azure za pośrednictwem interfejsu API Azure monitora. Te informacje jest używany do wykrywania malfeasance i spełnić wymagania zgodności.

[Przejdź do dokumentacji.][alertlogic-doc]

## <a name="appdynamics"></a>AppDynamics
AppDynamics zarządzania wydajności aplikacji (APM) umożliwia aplikacji właścicielom szybkie rozwiązywanie problemów z wąskich gardeł wydajności i optymalizacji wydajności swoje aplikacje uruchomione w środowisku platformy Azure. Można monitorować usługi w chmurze Azure (PaaS), role sieci web i proces roboczy, maszyn wirtualnych (IaaS), zdalny wykrywania usługi (Microsoft Azure Service Bus), Microsoft Azure kolejki, zdalnej usługi Microsoft Azure (Azure Blob), kolejek platformy Azure (Microsoft Service Bus), dane Magazynu i magazynu obiektów Blob platformy Microsoft Azure. AppDynamics APM jest dostępne w portalu Azure Marketplace.

[Przejdź do dokumentacji.][appdynamics-doc]

## <a name="circonus"></a>Circonus
Circonus jest mikrousług monitorowania i platforma do analiz w oparciu dla lokalnego lub SaaS wdrożenia. Jest w pełni automatable interfejsu API skoncentrowane na platformy jest bardziej skalowalny i monitoruje niezawodna niż systemów. Rozwinięte wymagania w zakresie opracowywania oprogramowania, Circonus zapewnia alerty na podstawie percentyl, wykresy, pulpity nawigacyjne i analizy uczenia maszynowego, które umożliwiają Optymalizacja biznesowa. Circonus monitoruje zasobów w chmurze Microsoft Azure i aplikacji w czasie rzeczywistym. Circonus służy do zbierania i śledzenia metryki dla zmiennych, które mają do pomiaru zasobów i aplikacji. Z Circonus uzyskasz systemowe wgląd w użycie zasobów platformy Azure, wydajność aplikacji i kondycji operacyjnej.

[Przejdź do dokumentacji.][circonus-doc]

## <a name="cloudhealth"></a>CloudHealth
Łączenia i zautomatyzować chmury przy użyciu platformy wbudowanej zaoszczędzić czas i pieniądze. CloudHealth zapewnia widoczność, intuicyjny optymalizacji i skale jednolite zarządzanie rozwiązaniami do zarządzania chmurą. Platforma Cloudhealth umożliwia przedsiębiorstwom i MSPs do zmaksymalizowania zwrotu z inwestycji w chmurze. Decyzje pewność wokół kosztów, użycie, wydajności i zabezpieczeń.

[Dowiedz się więcej.][cloudhealth-doc]

[Przejdź do dokumentacji.][cloudmonix-doc]

## <a name="cloudyn"></a>Cloudyn
Cloudyn zarządza i optymalizuje obejmującego wiele platform chmury hybrydowej, aby pomóc w pełni zdawać ich potencjalne chmury przedsiębiorstwa. Rozwiązanie SaaS zapewnia wgląd w użycie, wydajności i kosztów. Zawiera szczegółowe informacje i praktyczne zalecenia dotyczące inteligentne ładu optymalizacji i w chmurze. Cloudyn umożliwia odpowiedzialności za pośrednictwem dokładne obciążenia zwrotnego i hierarchicznych kosztów zarządzania alokacji. Cloudyn jest zintegrowany z monitorowania Azure szczegółowe informacje i zalecenia można wykonać w celu zoptymalizowania wdrożenia usługi Azure.

[Przejdź do dokumentacji.][cloudyn-doc]

## <a name="datadog"></a>Datadog
Datadog jest wiodących na świecie monitorowania usługa dla aplikacji w skali chmury. Stwarza razem danych z serwerów, baz danych, narzędzia i usługi do prezentowania ujednoliconego podglądu całego stosu. Te możliwości są dostarczane na platformie analizy danych w modelu SaaS. Usługa ta umożliwia deweloperów i zespoły Ops, aby umożliwić wspólną pracę, aby uniknąć przestoju, rozwiąż problemy z wydajnością i upewnij się, że programowanie i wdrożenie cykle zdążyć na czas. Dzięki integracji Datadog i na platformie Azure, można zbierać i Wyświetl metryki z całej infrastruktury. Korelowanie metryki maszyny Wirtualnej z metryki na poziomie aplikacji. Kątami Twoje metryki przy użyciu dowolnej kombinacji właściwości i znaczniki niestandardowe.

[Przejdź do dokumentacji.][datadog-doc]

## <a name="dynatrace"></a>Dynatrace
Dynatrace OneAgent integruje się z maszynami wirtualnymi Azure i usługi aplikacji za pośrednictwem mechanizmu rozszerzenie Azure. W ten sposób Dynatrace OneAgent zbierać metryki wydajności dotyczące hostów, sieci i usług. Oprócz wyświetlania tylko metryki, Dynatrace wizualizuje środowisk end-to-end. Przedstawia on transakcji po stronie klienta do warstwy bazy danych. Dynatrace zapewnia oparte na AI korelacji problemów i w pełni zintegrowana głównego Przyczyna analizy zapewniające wgląd poziomu metody kodu oraz bazy danych. To wgląd ułatwia optymalizację wydajności i rozwiązywanie problemów z znacznie.

[Przejdź do dokumentacji.][dynatrace-doc]

## <a name="opsgenie"></a>OpsGenie
OpsGenie działa jako dyspozytora alerty generowane przez platformę Azure. OpsGenie określa odpowiednich osób do powiadomienia, na podstawie na wywołanie harmonogramów i ich eskalacji. Go powiadamiać użytkowników przy użyciu za pośrednictwem poczty e-mail, wiadomości tekstowych (SMS), rozmowy telefoniczne lub wypychanie powiadomień. Platforma Azure generuje alerty dotyczące wykrytych problemów. OpsGenie gwarantuje, że pracujesz odpowiednich osób o problemie.

[Przejdź do dokumentacji.][opsgenie-doc]

## <a name="pagerduty"></a>PagerDuty
PagerDuty, wiodącego rozwiązania do zarządzania zdarzeniami, oferuje najwyższej jakości pomoc techniczną dla usługi Azure alertów na metryki. PagerDuty obsługuje powiadomienia na Azure monitorowanie alertów, powiadomień skalowania automatycznego zdarzenia z dziennika inspekcji i platformy poziomu metryki dla usług Azure. Nadaj te ulepszenia zwiększyć wgląd w podstawowej platformy Azure. Można w pełni korzystać PagerDuty jego możliwości zarządzania zdarzeniami dla odpowiedzi w czasie rzeczywistym. Rozwinięte integracja Azure jest możliwe za pomocą elementów webhook. Elementów Webhook pozwalają na konfigurowanie i dostosowywanie rozwiązanie szybkie i łatwe.

[Przejdź do dokumentacji.][pagerduty-doc]

## <a name="sciencelogic"></a>ScienceLogic
ScienceLogic zapewnia generacji IT usługi gwarancji platformy do zarządzania z dowolnego miejsca i technologii. ScienceLogic zapewnia skali, zabezpieczeń, automatyzacji i odporności konieczna uprościć zadania zarządzania, zasobów, usług i aplikacji. Platforma ScienceLogic używa interfejsów API usługi Azure na potrzeby interfejsu z Microsoft Azure. ScienceLogic zapewnia wgląd w czasie rzeczywistym do usług platformy Azure i zasobów. Aby dowiedzieć się, gdy coś nie działa i można go naprawić szybciej. Można również zarządzać Azure równolegle z innymi systemami centrum danych i chmur oraz usług.

[Dowiedz się więcej.][sciencelogic-doc]

## <a name="sumo-logic"></a>Logika sumo
Sumo jest bezpieczne, natywne chmury maszyny usługi analizy danych, dostarcza analizy w czasie rzeczywistym, ciągłe strukturalnych częściową strukturą i bez struktury danych w cyklu życia aplikacji i stosu. Więcej niż 1000 klientów na całym świecie polegają na Sumo logikę analizy i wgląd do kompilacji, uruchomione i zabezpieczania aplikacji oraz infrastruktury chmury. Z logiką Sumo klientów uzyskania nad wielodostępnej, model usług można zwiększyć konkurencyjnej, wartość biznesową i wzrost.

[Dowiedz się więcej.][sumologic-doc]
