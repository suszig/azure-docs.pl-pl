# <a name="introduction-to-azure-stack-networking"></a>Wprowadzenie do sieci Azure stosu

*Dotyczy: Azure stosu zintegrowanych systemów i Azure stosu Development Kit*

Stos Azure oferuje szeroką gamę możliwości sieciowych, które mogą być używane razem lub oddzielnie:
- Łączność między zasobami Azure stosu: łączenie zasobów platformy Azure w bezpieczny, prywatnej sieci wirtualnej w chmurze.
- Połączenie z Internetem: komunikowanie się z zasobami Azure stosu i w Internecie.
- Połączenie lokalne: łączenie sieci lokalnej z zasobami Azure stosu za pośrednictwem wirtualnej sieci prywatnej (VPN) przez Internet lub przez dedykowane połączenie stos Azure.
- Kierunek ruchu i równoważenia obciążenia: obciążenia równoważenie ruchu do serwerów w tej samej lokalizacji i bezpośredniego ruch do serwerów w różnych lokalizacjach.
- Zabezpieczenia: filtrowania ruchu sieciowego między podsieciami sieci lub poszczególnych maszyn wirtualnych (VM).
- Routing: korzystać z routingu domyślne lub routingu między stosu Azure i zasobów lokalnych w pełni kontrolować.
- Możliwości zarządzania: monitorowanie i zarządzanie nimi stosu Azure zasobów sieciowych.
- Narzędzia wdrażania i konfiguracji: portalu sieci web lub narzędzia wiersza polecenia i platform wdrażania i konfigurowania zasobów sieciowych.
