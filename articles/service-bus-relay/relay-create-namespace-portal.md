# <a name="create-a-relay-namespace-using-the-azure-portal"></a>Tworzenie przestrzeni nazw usługi Relay za pomocą witryny Azure Portal
Przestrzeń nazw jest wspólnym kontenerem dla wszystkich składników usługi Azure Relay. W jednej przestrzeni nazw może znajdować się wiele usług Relay, a przestrzenie nazw często pełnią rolę kontenerów aplikacji. Obecnie istnieją dwa różne sposoby tworzenia przestrzeni nazw usługi Relay.

1. Azure Portal (w tym artykule)
2. Szablony usługi [Azure Resource Manager](../azure-resource-manager/resource-group-overview.md)
