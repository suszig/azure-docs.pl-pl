# <a name="how-to-log-a-security-event-support-ticket"></a>Jak rejestrować biletu pomocy technicznej zdarzeń zabezpieczeń

1. Przejdź do [Portal wydawcy](https://support.microsoft.com/en-us/getsupport?wf=0&tenant=ClassicCommercial&oaspworkflow=start_1.0.0.0&locale=en-us&supportregion=en-us&pesid=16230&ccsid=636450758943226673) i zaloguj się przy użyciu poświadczeń firmy Microsoft.
2. Wybierz "Zdarzeń zabezpieczeń" jako typ problemu i wybrać kategorii "Zdarzenia zabezpieczeń" i "Luka w zabezpieczeniach".

    ![Typ zdarzenia i definicji][1]

3. Wybierz typ problemu i kategoria, kliknij "**żądanie uruchomienia**" przycisku.

    i. Co to jest problem i/lub luk w zabezpieczeniach.

    II. Dla luk w zabezpieczeniach Dodaj CVE (mitre.org) lub wypełniony limit CVSS3 v3 kalkulatora (https://www.first.org/cvss/calculator/3.0).

    III. Istnieje już rozpoznawania lub migracji? Jeśli tak, następnie podaj czynności korygujące.

    IV. Masz komunikat, który chcesz wysłać do klientów? Firma Microsoft będzie współpraca z Tobą sformułować odpowiedni komunikat, jeśli ma to zastosowanie.

4. Potwierdzenie przesyłania — po przesłaniu problemu, firma Microsoft będzie otrzymanie w ciągu jednego dnia roboczego i przypisać problem priorytetu i wagi.

    - Jeśli potrzebujesz do komunikowania się z nami o problem, użyj numeru potwierdzenia w korespondencję.
    - Postęp można wyświetlać na problem w dowolnym momencie.

5. Co dalej? W zależności od tego problemu i ważność mogą zostać podjęte następujące kroki:

    - Firma Microsoft będzie komunikować się na wynik naszych oceny do Ciebie. W zależności od wyniku możemy usunąć lub żądania zmodyfikowanie Twojej oferty. W takim przypadku firma Microsoft będzie działać z Tobą w celu zapewnienia zminimalizowane przerw w działaniu wpływ na klientów.
    - Firma Microsoft będzie działać z Tobą w celu zmniejszenia wpływu zdarzenia/lukę w zabezpieczeniach dla naszych klientów wzajemnego.
